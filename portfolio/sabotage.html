Philadelphia

Posters & Postcards

Project Overview

The original goal of this project was to create three postcard designs for Philadelphia, showcasing a landmark. I made this project for F.I.R.S.T. Institute in 102 and revisited it in 108 to add another location and designs for the back of the postcards.

Research and Specifications

I began this project by researching famous historical landmarks in Philadelphia. I collected royalty free images of the landmarks I wanted to showcase.

First Draft

My first draft of this project included three postcards for Independence Hall. I chose the color palettes using the color guide panel in Adobe Illustrator.

Work in Progress

For the second round of postcards, I decided to showcase the Betsy Ross House. I found three royalty free images of the building and began drawing an illustrated version.

Obstacles and Solutions

Since I decided to use complementary color palettes for these designs, I had a hard time choosing colors for some of the postcards. I was able to work though this issue by using the recolor option in the color guide panel to quickly test out different color combinations.

Critique and Revisions

Since the original version of this project only required a design for the front of the postcard, I decided to create designs for the back of the postcard as well.

Next Steps

To wrap up this project, I created mockups for the posters and postcards.

Final Product and Mockups

The final product includes six postcard/poster designs, with the postcards having both a front and back design. Three of the designs feature Independence Hall and the other three feature the Betsy Ross House.
